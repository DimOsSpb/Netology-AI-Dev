# Решение
- В решении я не использовал dotenv, а применил pydantic settings, который позволяет работать с настройками и env с более гибкой и расширенной функциональностью. В частности у меня ключи api лежат в файлах и в [config.py](config.py) дополнительно заложена работа с файлами - как значениями ключей из env. Список провайдеров и настройка провайдера по умолчанию также реализована в Settings(BaseSettings).
- [Пример .env.example](.env.example)
